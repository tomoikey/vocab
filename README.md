# How to run
```shell
❯ vocab --help
Usage: vocab [OPTIONS]

Options:
  -f, --file <FILE>  単語ファイルのパス (JSON形式) [default: words.json]
  -h, --help         Print help
  -V, --version      Print version
```

# How to install
```shell
❯ cargo install --release
❯ cp ./target/release/vocab /usr/local/bin
❯ vocab --version
```
